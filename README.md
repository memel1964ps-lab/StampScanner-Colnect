# StampScanner Colnect

**App open-source de reconhecimento de selos por foto usando IA + Colnect CAPI**

Um aplicativo gratuito para Android e iOS que permite a qualquer colecionador fotografar um selo e receber automaticamente todas as informações completas do catálogo Colnect.

## Funcionalidades planejadas
- Reconhecimento de selos por câmera usando IA (on-device)
- Busca automática de dados completos via **Colnect CAPI** (descrição, variantes, raridade, valores estimados, etc.)
- Suporte total ao português (Brasil)
- Modo offline com cache local
- Sugestões de trocas com outros colecionadores
- Sistema fácil de contribuição (envio de fotos e correções para o catálogo)

## Status do projeto

## Tecnologias
- Flutter (Dart) – app cross-platform
- TensorFlow Lite / MediaPipe – reconhecimento de imagem local
- Colnect CAPI – dados oficiais do catálogo
